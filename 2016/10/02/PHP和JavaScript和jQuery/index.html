三、DOM(Document、Object、Model)：

  • node（节点），其实不单单只有这3种节点，总共有12种：

    1. element node（元素节点）：p，div；
    2. text node（文本节点）：标签里面的文字；
    3. attribute node（属性节点）：例如id、class里面的值；

  • 元素节点的操作（这些方法都是document对象的。）：

    1. getElementById("IDName")：
    2. getElementsByTagName("TagName"):获取到的是对象数组.
    3. getElementsByClassName("ClassName"):获取到的是对象数组.

  • 属性节点的操作（以下方法都是元素节点的。不能通过document调用）：

    1. getAttribute("AttributeName")：获取属性.
    2. setAttribute(Attribute, NewValue)：设置属性.

    • element.childNodes:获取Element Node的子节点数组。
    • node.nodeType:获取node的类型。返回1，2，3。。。
