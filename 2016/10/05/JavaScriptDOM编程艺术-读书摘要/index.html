JavaScriptDOM编程艺术 读书摘要

站在巨人的肩膀上

一、Node

  • element node:元素节点
  • text node:文本节点
  • attribute node:属性节点

二、获取 Element Node：

下述3个方法不仅可以作用于document也可以作用于element node对象。即组合使用

  • document.getElementById(id):返回一个对象。
  • document.getElementsByTagName(tag):返回一个数组。
  • document.getElementsByClassName(class):返回一个数组。

三、获取 Element Node 的子节点：

  • Element_Node.childNodes：返回全部子节点的数组。
  • Node_Object.nodeType:每个node都有一个nodeType属性。属性值有12种。
  • 1表示element node，2表示attribute node，3表示Text node。。。
  • Element_Node.firstChild:返回第一个子节点。
  • Element_Node.lastChild:返回最后一个子节点。

四、获取和设置 Attribute Node：

下述2个function仅仅能作用于元素节点对象（Object Of Element Node）

  • Element_Node.getAttribute(attribute):
  • Element_Node.setAttribute(attribute,newValue):

五、获取、设置 Text Node的值：

  • var value = Element_Node.childNodes[0].nodeValue:因为文本节点是包含在元素节点里面的，其实是元素节点的子节点。value = 'new text..'就是重现赋值了。
  • 元素节点对象.nodeName:获取元素节点名称。
  • 元素节点对象.nodeType:获取元素节点类型1或2或3.。。。

六、创建标签：

  • document.write("<p></p>"):不用了。
  • Element_Node.innerHTML: 用来获取和设置整段HTML代码。
  • document.createElement(tag):创建Element Node。
  • document.createTextNode(text):创建Text Node。
  • parentElementNode.appendChild(Element_Node):把新元素节点插入到父元素节点的最后一个子元素后面。
  • parentElement.insertBefore(newElement, targetElement)：把新元素节点插入到目标节点的前面。其实parentElement就是targetElement.parentNode。
